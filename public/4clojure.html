1. Nothing but the Truth

This is a clojure form. Enter a value which will make the form evaluate to true. 

(defn nothing-but-truth []
  (= true true))

2. Simple Math

If you are not familiar with polish notation, simple arithmetic might seem confusing. 

(defn simple-math []
  (= (- 10 (* 2 3)) 4))

3. Intro to strings

Clojure strings are Java strings. This means that you can use any of the Java string methods on Clojure strings. 

(defn intro-to-strings []
  (= "HELLO WORLD" (.toUpperCase "hello world")))

4. Intro to lists

Lists can be constructed with either a function or a quoted form. 

(defn intro-to-lists []
    (= (list :a :b :c) '(:a :b :c)))

5. Lists: conj

When operating on a list, the conj function will return a new list with one or more items “added” to the front. 

(defn lists-conj []
  (= '(1 2 3 4) (conj '(2 3 4)))
  (= '(1 2 3 4 (conj '(3 4) 2 1))))

6. Intro to vectors

Vectors can be constructed several ways. You can compare them with lists. 

(defn intro-to-vectors []
  (= [:a :b :c] (list :a :b :c) (vec '(:a :b :c)) (vector :a :b :c)))

7. Vectors: conj

When operating on a Vector, the conj function will return a new vector with one or more items “added” to the end. 

(defn vectors-conj []
  (= [1 2 3 4] (conj [1 2 3] 4))
  (= [1 2 3 4] (conj [1 2] 3 4)))

conj is a clipping, (technically apocope) of conjoin.

8. Intro to sets

Sets are collections of unique values. 

(defn intro-to-sets []
  (= #{:a :b :c :d} (set '(:a :a :b :c :c :c :c :d :d)))
  (= #{:a :b :c :d} (clojure.set/union #{:a :b :c} #{:b :c :d})))

In set theory, the union of a collection of sets is the set of all elements in the collection. - Union (set theory), Wikipedia

In the second example above, there are two sets in the collection.

  1. #{:a :b :c}
  2. #{:b :c :d}

The union of the sets A and B (A âˆª B) is the set #{:a :b :c :d}.

9. Sets: conj

When operating on a set, the conj function returns a new set with one or more keys “added”. 

(defn sets-conj []
  (= #{1 2 3 4} (conj #{1 4 3} 2)))

10. Intro to maps

Maps store key-value pairs. Both maps and keywords can be used as lookup functions. Commas can be used to make maps more readable, but they are not required. 

(defn intro-to-maps []
  (= 20 ((hash-map :a 10 :b 20 :c 30) :b))
  (= 20 (:b {:a 10 :b 20 :c 30})))

Write a function which returns the total number of elements in a sequence.

Special Restrictions: count

Iteration 1 

(defn count-a-sequence [s]
  (reduce (fn [total item]
            (inc total)) 0 s))

Other solutions 

(defn count-a-sequence [s]
  (reduce + (map (constantly 1) s)))

28. Flatten a sequence

  • Difficulty: Easy
  • Topics: seqs core-functions
  • Effort: ðŸ§ ðŸ§ ðŸ§ 

Write a function which flattens a sequence.

Special Restrictions: flatten

Iteration 1 

(defn my-flatten [coll]
  (seq (reduce (fn [result item]
            (if (coll? item)
                (into result (my-flatten-1 (first (list item))))
                (conj result item)))
          []
          coll)))

Iteration 2 

(defn my-flatten [item]
  (cond
    (coll? item) (mapcat my-flatten-2 item)
    :else [item]))
